GameScope: Predicting Video Game Success from Market and Review Data

In this project, I will analyze which factors are related to the success of video games and whether game success can be predicted using market and review-based features in order to find best game for us. The motivation behind this topic is that video games differ greatly in terms of genre, price, release year, and user reception, and these differences may help explain why some games become more successful than others.

The main data source will be a publicly available video game dataset obtained from Kaggle. The dataset is expected to include variables such as game title, genre, price, release date, user ratings, review counts, and other game-related features. Since the project requires enrichment when using public data, I plan to extend the dataset by creating additional variables such as game age, genre count, free-to-play status, and grouped success levels based on review or rating measures.

The data will be collected by downloading CSV files from public sources and loading them into Python with pandas. After collecting the data, I will perform preprocessing steps such as cleaning missing values, removing duplicates, and selecting the most relevant variables for analysis. The dataset is expected to contain several thousand observations and a mix of numerical and categorical variables.

The project will use exploratory data analysis, visualizations, hypothesis testing, and machine learning methods to identify important predictors of success and evaluate whether video game success can be predicted from the available features.
